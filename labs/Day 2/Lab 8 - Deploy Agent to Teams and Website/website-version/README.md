# Marina Trust Bank Website

This website is the customer-facing front door for Lab 8. It sends application
JSON to an ordinary Power Automate HTTP flow. Power Automate conditions decide
the outcome, email the user a safe summary, and return JSON for the website to
render. This **website path** does not use a Copilot Studio agent; Lab 8 Part A
provides a separate informational agent in Microsoft Teams.

Labs 9 and 10 reuse the Marina Trust website scenario through the Copilot
Studio **Demo website** or **Custom website** channel. In those labs, the
embedded chatbot—not this standalone HTML form—triggers the agent or prompt
flow.

## Run locally

```bash
cd "labs/Day 2/Lab 8 - Deploy Agent to Teams and Website/website-version"
python3 -m http.server 8000
```

Open `http://localhost:8000`, paste the HTTP POST URL generated by **When an
HTTP request is received**, choose a trainer test case, enter your own email
address, and submit.

The endpoint is stored only in browser `localStorage`. Do not place a real flow
URL in `script.js` or commit it to source control.

See the parent [Lab 8 guide](../index.md) for the complete Power Automate
configuration.
